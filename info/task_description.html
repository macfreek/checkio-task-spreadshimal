The robots Sophia and Nikola are engaged in their favourite pastime, reviewing programs. Sophia is looking at the columns in a spreadsheet program.

A B C D ... X Y Z AA AB AC ... AY AZ BA BB ... ZY ZZ AAA AAB ...

"This looks like a hexavigesimal base system"

"A what?," replied Nikola.

"A base-26 numeral system. You know, like binary (base-2), decimal (base-10) or hexadecimal (base-16). Remember from The Good Radix? There are 26 symbols here, so this is a hexavigesimal numeral system!"

Nikola looks carefully at the list. "I don't think so. If A is a 0, what is AA doing after the Z? I think A is 1, and Z is 26. So this must be a heptavigesimal numeral system, and the 0 at the start is not shown."

"That can't be right either! If this is a heptavigesimal numeral system, the symbol after the Z should be A0 instead of AA. I think it's neither a hexavigesimal nor a heptavigesimal numeral system. It's a spreadshimal numeral system!"

Your task is to implement the spreadshimal numeral system. Given a number, return the label on the nth column in the spreadsheet.

Input: One arguments. An integer n.

Output: A string. The representation of nth spreadshimal in capital letters. If n is zero, an empty string. If n < 0, the nth spreadshimal number prepended with a minus symbol ('-').

Example: 

checkio(1) == 'A'
checkio(2) == 'B'
checkio(26) == 'Z'
checkio(27) == 'AA'

How it is used: spreadshimal numbers are used in columns of spreadsheets, and may teach you about non-standard numeral systems.
